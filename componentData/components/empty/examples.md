## Empty 组件示例

### basic-usage



```vue
<template>
  <fin-empty description="description" />
</template>
```

### custom-image



```vue
<template>
  <fin-empty
    image="https://shadow.elemecdn.com/app/element/hamburger.9cf7b091-55e9-11e9-a976-7f4d0b07eef6.png"

```

### image-size



```vue
<template>
  <fin-empty :image-size="200" />
</template>
```

### bottom-content



```vue
<template>
  <fin-empty>
    <fin-button type="primary">Button</fin-button>
  </fin-empty>
</template>
```
