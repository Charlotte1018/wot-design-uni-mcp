# Progress 进度条

用于展示操作的当前进度。

## 基本用法

设置百分比 `percentage`。

## 隐藏进度文字

设置 `hide-text` 隐藏进度文字。

## 设置状态

设置 `status`，告知用户当前状态和预期。

## 修改颜色

设置 `color` 修改进度条颜色。

`color` 也可以设置为数组或者函数。数组如果只传入颜色，则自动计算每个颜色的进度边界。函数需要返回一个颜色值。
