# Backtop 回到顶部  

用于返回页面顶部的操作按钮。

## 基本用法

由于返回顶部需要实时监听滚动条位置，但是在uniapp的组件中不能获取页面的滚动信息，
所以只能在页面的`onPageScroll`生命周期中获取滚动条位置，再通过`Props`传递给组件。
