# IndexBar 索引栏

用于列表的索引分类显示和快速定位。

## 基本用法

使用一个固定高度的元素包裹`wd-index-bar`组件,组件的宽高会和包裹元素相同。

将`wd-index-anchor`作为子组件使用,会根据 anchor 组件的`index`属性生成锚点以及侧边栏。
