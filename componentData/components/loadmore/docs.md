# loadmore 加载更多

用于在列表底部展示加载状态。

## 基本用法

在需要进行加载的列表的底部引入该组件即可。当滑动到列表底部时，通过设置`state`展示不同的文案。

## 自定义文案

通过设置`loading-text`、`finished-text`、`error-text`配合`state`展示不同状态时的文案

## 点击继续加载

当 state 为 error 时，点击文案，组件会触发`loadmore`事件

## 应用实现

配合`onReachBottom`事件实现滚动到底部加载更多
