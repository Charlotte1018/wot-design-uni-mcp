#  Pagination 分页

当数据量过多时，使用分页分解数据。

## 基本用法

通过 `v-model` 来绑定当前页码，`total`设置总条数，`page-size`设置一页展示条数，默认为10条，总页数通过`total`和`page-size`自动计算。

## Icon图标

设置 `show-icon` 属性，将分页导航展示为Icon图标。

## 文字提示

设置 `show-message` 属性，展示文字提示。
